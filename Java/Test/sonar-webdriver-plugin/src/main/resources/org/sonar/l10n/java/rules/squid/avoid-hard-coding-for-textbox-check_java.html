Avoid hardcoding in sendkeys, either read data from cucumber or data xml.

Noncompliant Code Example

      public void type() {
    	searchBox.sendKeys("typeMe"); // Noncompliant 
    }

    
    public void type1() {
    	searchBox.type("typeMe"); // Noncompliant 
    }

Compliant Solution

  Either pass the data to user code from cucumber or read from data.xml
