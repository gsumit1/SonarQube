WebDriver commands should not be used in tests. Best practice is to have WebDriver commands in Base or Page Objects.

Noncompliant Code Example

  By.cssSelector("#form div input"); // Noncompliant

Compliant Solution

  // Implement wrapper methods in a Base class, for example:
  public void type(By locator, String inputText) {
        find(locator).sendKeys(inputText);
  }

  private WebElement find(By locator) {
        return driver.findElement(locator);
  }
