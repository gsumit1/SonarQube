Avoid using hardcoding in dropdown, either read data from cucumber or data xml.

Noncompliant Code Example

      public void dropdown() {
    		dropdown.selectByVisibleText("CINGULAR WIRELESS (4410139)"); // Noncompliant 
    }

    
    public void dropdown1() {
    	dropdown.selectByValue("typeMe"); // Noncompliant 
    }
    
    public void dropdown3() {
    	dropdown.selectByIndex(1); // Noncompliant 
    }

Compliant Solution

  Either pass the data to user code from cucumber or read from data.xml
