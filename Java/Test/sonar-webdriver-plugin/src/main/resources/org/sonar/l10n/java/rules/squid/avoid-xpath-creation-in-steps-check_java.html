Avoid xpath creation for web element instead use @FindBy in page class. For dynamic xpath creation you can use but in page class only

Noncompliant Code Example

 public void dropdown() {
		getDriver().findElement(By.xpath("//div[1]"); // Noncompliant 
	}

	public void dropdown1() {
		getDriver().findElements(By.xpath("//div"); // Noncompliant 
	}

Compliant Solution

  use @FindBy in page class.
